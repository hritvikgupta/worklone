# Quick Start

Getting Started with Workdocs

## Introduction

Workdocs is a lightweight documentation surface designed for product and engineering teams.

This page is currently **dummy content** loaded from markdown so you can replace it later with your real docs.

## Terminology

- **Workspace**: Your project-level container.
- **Pages**: Individual markdown-backed documentation pages.
- **Navigation**: Grouped links shown in the sidebar.
- **On this page**: Section anchors generated from headings.

## Automatic Installation

1. Create your docs files in a markdown folder.
2. Connect your content source to the docs frontend.
3. Publish and verify links in the sidebar and table of contents.

## Enjoy

You now have a documentation shell with:

- Left sidebar navigation
- Main markdown-rendered article area
- Right-side table of contents
